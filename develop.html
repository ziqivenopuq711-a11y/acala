Developer Guide

Build on Acala using Substrate tooling or Ethereum-compatible libraries via EVM+.

Installation & Setup

Substrate (Polkadot.js)

The official JavaScript/TypeScript library for interacting with Substrate-based chains including Acala. Supports all Substrate RPC methods, extrinsic signing, and storage queries. 

npm install @polkadot/api
npm install @polkadot/keyring
npm install @polkadot/util

EVM+ (Ethereum Tools)

Use standard Ethereum libraries to deploy and interact with Solidity contracts on Acala's EVM+ layer. ethers.js v6 is recommended for new projects. 

npm install ethers
# or legacy web3.js
npm install web3

# Hardhat for contract development
npm install --save-dev hardhat

Connecting to Acala

WebSocket Connection (Recommended)

WebSocket connections are preferred for most use cases as they support subscriptions (new blocks, storage changes, events) and maintain persistent state. Replace YOUR_API_KEY with your key from dashboard.dwellir.com. 

const { ApiPromise, WsProvider } = require('@polkadot/api');

async function main() {
  const provider = new WsProvider('wss://api-acala.n.dwellir.com/YOUR_API_KEY');
  const api = await ApiPromise.create({ provider });

  const chain = await api.rpc.system.chain();
  const version = await api.rpc.state.getRuntimeVersion();

  console.log('Connected to:', chain.toString());
  console.log('Spec version:', version.specVersion.toNumber());
  // Expected: Acala, 2300

  await api.disconnect();
}

main().catch(console.error);

HTTPS Connection

Use HTTPS for one-shot queries where a persistent connection is not required, such as reading a single storage value or submitting a pre-signed extrinsic. 

const { ApiPromise, HttpProvider } = require('@polkadot/api');

const provider = new HttpProvider('https://api-acala.n.dwellir.com/YOUR_API_KEY');
const api = await ApiPromise.create({ provider });

const blockHash = await api.rpc.chain.getFinalizedHead();
console.log('Finalized head:', blockHash.toHex());

EVM+ Connection with ethers.js

To interact with Acala's EVM+ layer, use a standard Ethereum JSON-RPC provider pointed at the Acala EVM+ endpoint. MetaMask can also be configured to use this endpoint. 

const { ethers } = require('ethers');

// EVM+ endpoint for Acala
const provider = new ethers.JsonRpcProvider(
  'https://eth-rpc-acala.aca-api.network'
);

const blockNumber = await provider.getBlockNumber();
console.log('EVM+ block number:', blockNumber);

// Deploying a contract (Hardhat config)
// hardhat.config.js
module.exports = {
  networks: {
    acala: {
      url: 'https://eth-rpc-acala.aca-api.network',
      chainId: 787,
      accounts: [process.env.PRIVATE_KEY]
    }
  }
};

Interacting with DeFi Pallets

Querying aUSD Balance

aUSD is tracked via the tokens pallet using a CurrencyId of { Token: 'AUSD' }. 

// Query aUSD balance for an address
const address = '23M5ttkmR6Kco6wSSGtxKDM5gbeXqAXbBRdCy4m2VHNnJPFP';
const currencyId = { Token: 'AUSD' };

const balance = await api.query.tokens.accounts(address, currencyId);
console.log('aUSD free balance:', balance.free.toString());

LDOT Liquid Staking

// Mint LDOT by staking DOT via the homa pallet
const dotAmount = api.createType('Balance', '1000000000000'); // 1 DOT

const tx = api.tx.homa.mint(dotAmount);

// Sign and send
await tx.signAndSend(account, ({ status }) => {
  if (status.isInBlock) {
    console.log('Mint LDOT included in block:', status.asInBlock.toString());
  }
});

DEX Token Swap

// Swap ACA for AUSD via the built-in DEX
const path = [{ Token: 'ACA' }, { Token: 'AUSD' }];
const supplyAmount = api.createType('Balance', '1000000000000');
const minTargetAmount = api.createType('Balance', '900000000000');

const swapTx = api.tx.dex.swapWithExactSupply(
  path,
  supplyAmount,
  minTargetAmount
);

await swapTx.signAndSend(account);

Development Tools

Polkadot.js Apps

The browser-based Substrate UI. Connect to the Dwellir WSS endpoint to inspect storage, submit extrinsics, and browse on-chain events. Available at polkadot.js.org/apps.

Subxt (Rust)

Type-safe Rust client for Substrate chains. Generate bindings from Acala's metadata and interact with pallets in strongly-typed Rust code. Ideal for high-performance off-chain workers and indexers. 

[dependencies]
subxt = "0.34"

py-substrate-interface

Python library for interacting with Substrate chains. Supports all RPC methods, extrinsic construction, and SCALE encoding/decoding. Good for scripting, analytics, and data pipelines. 

pip install substrate-interface

Dwellir

Managed RPC infrastructure for Acala Network.
