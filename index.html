Acala Network — DeFi Hub of Polkadot

Acala is the decentralised finance hub and stablecoin platform of the Polkadot ecosystem, operating as parachain ID 2000 on the Polkadot relay chain. It provides foundational DeFi primitives: the aUSD decentralised stablecoin, LDOT liquid staking, and a built-in DEX — all secured by Polkadot's shared security model. The native token ACA powers governance, fee payment, and incentive mechanisms. Acala's EVM+ layer enables Ethereum-compatible smart contracts alongside Substrate pallets.

Core Features

🪙 DeFi Primitives

  • aUSD — decentralised multi-collateral stablecoin pegged to the US dollar, minted via over-collateralised vaults
  • LDOT — liquid staking token representing staked DOT, retaining liquidity while earning staking rewards
  • Built-in DEX — automated market maker enabling trustless, permissionless token swaps entirely on-chain

🔗 Polkadot Shared Security

  • Parachain ID 2000 on the Polkadot relay chain with full shared security
  • Block finality guaranteed by Polkadot validators, removing the need for a separate validator set
  • XCM cross-chain messaging enables seamless interoperability with all connected parachains

⚡ EVM+ Compatibility

  • Ethereum-compatible execution environment built directly on the Substrate runtime
  • Deploy Solidity contracts using ethers.js, web3.js, or Hardhat
  • Connect with MetaMask and other EVM-compatible wallets with no extra configuration
